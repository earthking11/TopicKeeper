# 📄 PDF 精准语义裁剪工具 (PDF Semantic Whiteout Tool)

> 基于本地大模型（LLM）的 PDF 扫描件智能清洗工具，让文档脱敏与整理更高效。

## ✨ 项目简介

面对几十页的 PDF 会议纪要扫描件，如果你只需要分享其中某一个特定的议题，传统的手动编辑既耗时又容易破坏排版。

本工具利用 **本地 OCR** 和 **本地 LLM** 技术，能够精准识别文档结构。只需输入一个关键词，程序即可自动保留“纪要开头”、“目标议题”和“纪要结尾”，并将其他无关议题内容进行**语义涂白**，全程无需联网，数据不出本地，安全无忧。

## 🛠 核心功能

*   **精准语义理解**：不依赖简单的关键词匹配，而是利用 LLM 理解议题上下文，精准定位目标内容。
*   **结构化保护**：智能识别并保护“纪要开头”（Header）、“纪要结尾”（Footer，如出席名单、落款），确保文档完整性。
*   **原始排版保留**：采用坐标级涂白技术，不进行页面重组，完美保留原始 PDF 的扫描件风格和格式。
*   **隐私安全优先**：所有计算（OCR 分析、语义判断）均在本地完成，无需上传云端，非常适合处理敏感文件。

## 🧰 技术栈

*   **OCR 引擎**: `RapidOCR` (基于 ONNXRuntime) - 高效的中文文本检测与识别。
*   **PDF 处理**: `PyMuPDF` (fitz) - 高性能的 PDF 渲染与修改。
*   **AI 模型**: `Local LLM` (兼容 LM Studio / Ollama) - 负责文档结构分析与意图识别。
*   **开发语言**: Python 3.x

## 🚀 快速开始

1.  安装依赖：
    ```bash
    pip install pymupdf rapidocr_onnxruntime openai
    ```
2.  启动 LM Studio 并加载模型（如 Qwen2.5-7B-Instruct），开启 Server。
3.  运行脚本并输入关键词。
